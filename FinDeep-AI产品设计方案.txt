🧠 产品设计方案：「FinDeep」（金融深研）— AI多智能体深度研报生成系统
================================================================================

一、产品定位
-----------
一句话描述：面向个人投资者和独立分析师的AI驱动、多智能体协作的深度金融研究报告自动生成平台。

核心价值：传统金融研报依赖大量人工检索、数据整理和主观判断，个人投资者难以获得机构级别的深度分析。FinDeep通过多智能体对抗性辩论 + RAG事实锚定，自动化生成平衡、可追溯、有深度的研究内容。


二、核心创新点（区别于市面产品）
--------------------------------

创新1：多智能体"对抗性辩论"架构 ⭐

传统AI写作是单模型直接生成结论，容易产生偏见和幻觉。FinDeep独创性地引入三角色辩论机制：

┌─────────────────────────────────────────────────────────────┐
│                    FinDeep 多智能体架构                       │
│                                                             │
│   📊 多方分析师                    🐻 空方分析师               │
│   (Bull Agent)                    (Bear Agent)              │
│   寻找投资机会                      寻找风险隐患                │
│        │                               │                     │
│        └───────────┬───────────────────┘                     │
│                    │ 观点碰撞                                │
│                    ▼                                         │
│            🔍 事实核查员 (Fact-Checker)                       │
│            逐一验证双方引用的数据和事实                         │
│                    │                                         │
│                    ▼                                         │
│            📝 综合编辑 (Synthesizer)                          │
│            融合多方观点，生成平衡研报                           │
│                    │                                         │
│                    ▼                                         │
│            📄 最终研报 + 可溯源引用                            │
└─────────────────────────────────────────────────────────────┘

为什么创新：这是将投资领域的"多空博弈"思维产品化——不是让AI给答案，而是让AI模拟真实的投资辩论过程，产出更接近机构分析师水平的平衡性研究。


创新2："溯源锚定"引文系统 ⭐

大多数AI生成内容无法追溯信息来源。FinDeep的每一句分析性结论都强制关联到RAG检索到的具体文档段落：

- 用户点击任何分析文字，右侧面板自动高亮原文出处
- 事实核查Agent会标记"无可靠信源支撑的推断"，用⚠️警示
- 支持上传自有文档（财报PDF、会议纪要等）作为私有知识库


创新3："事件链"时间推理引擎 ⭐

不仅分析"当前状态"，还自动构建事件时间链：

营收下降 12% ──→ 管理层变更 ──→ 战略转向公告 ──→ 机构评级下调
  (Q2财报)       (Q3公告)        (Q3电话会)        (Q4研报)

AI自动识别事件之间的因果关系，生成"叙事链"可视化时间轴


创新4：渐进式深度钻取交互

不是一次性输出长篇报告，而是：
1. 输入公司/行业 → 生成概览摘要（1分钟阅读）
2. 点击感兴趣的点 → 展开详细分析（带数据图表）
3. 对分析有疑问 → 追问Agent进行深度挖掘
4. 信任但想验证 → 一键切换Bull/Bear视角看同一问题的不同解读

六、和市面上产品的差异化
-------------------------

vs ChatGPT + 联网搜索：
  不是给答案，而是展示多元观点博弈过程 + 可溯源

vs 传统研报工具（Wind、同花顺）：
  自动化 + 自然语言交互 + 个人可负担

vs Perplexity / AI Search：
  领域深度（金融专有数据 + 结构化分析框架）远超通用搜索

vs 现有的AI研报产品：
  对抗性辩论架构是独有创新，避免AI"一边倒"的幻觉问题
